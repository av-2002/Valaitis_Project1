Art Spring

This assignment was my first large website design. The premise of the assignment was to redesign and take an already built website and redesign it to feel more cohesive and aesthetically pleasing. Keeping with the theme of the Art Spring Festival as well.

Process

IA Diagram

The first part of this assignment was gathering the already included information from the website and having people sort cards to fit the navigation section. Each person sorted the same cards and from there I created flow charts from each persons results.

Style Tiles

From there we created 3 style tiles to create an aesthetic for the Art Spring Site

Wireframes

With the style tiles picked we then made 5 Wireframes that then we turned into mockups.

User Testing

After Creating the website we moved to user Testing, I tested the website twice and asked a series of questions that the user would then follow and complete the task asked of them. The results were then recorded and changes made to account for accessibility and presentation.
